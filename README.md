# GitHub Explorer

A lightweight web application for exploring GitHub users, organizations, repositories, topics, and developer resources through the GitHub REST API.

GitHub Explorer provides an intuitive interface for searching GitHub content, viewing developer statistics, analyzing repositories, and discovering useful open-source projects and learning resources.

---

## Features

- Search GitHub users
- Search GitHub organizations
- Explore repositories
- Browse GitHub topics
- Discover developer books and educational resources
- View detailed developer statistics
- Repository analytics
- Weekly commit activity charts
- Repository star distribution
- Top repositories overview
- Random repository and developer discovery
- Dark and Light themes
- Responsive design
- GitHub REST API integration

---

## Developer Statistics

GitHub Explorer can display detailed information for GitHub users, including:

- Public repositories
- Followers and following
- Total repository stars
- Total forks
- Commit statistics
- Weekly activity charts
- Top repositories
- Repository overview

## Installation

Clone the repository:

```bash
git clone https://github.com/Yrashka200/github-explorer.git
```

Open the project folder:

```bash
cd github-explorer
```

Launch the application by opening the `index.html` file in your web browser.

No installation, build tools, or additional dependencies are required.

---

## Usage

1. Enter a GitHub username, repository name, or keyword.
2. Select a search category.
3. Browse the results.
4. Click on a GitHub user to view detailed statistics and charts.
5. Use the **Random** button to discover new developers and repositories.

---

## Search Categories

- All
- Users
- Organizations
- Topics
- Repository Collections
- Learning Resources

---

## API

GitHub Explorer uses the official GitHub REST API.

Please note that anonymous API requests are subject to GitHub rate limits.
